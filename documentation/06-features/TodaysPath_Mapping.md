# TODAY'S PATH - MOOD-BASED CONTENT MAPPING

**Purpose:** Provide personalized spiritual guidance based on user's emotional state.

**How it works:**
- User selects mood during onboarding or weekly check-in
- Mood is displayed on Home screen (top section)
- Today's Path shows 3 personalized suggestions:
  1. Daily Verse (Qur'an)
  2. Suggested Dua
  3. Recommended Story (Prophet)

---

## IMPLEMENTATION LOGIC

### Data Structure

```typescript
// types/mood.ts
export type MoodType = 'anxious' | 'seeking' | 'grateful' | 'weary';

export interface TodaysPathContent {
  verse: {
    surahName: string;
    verseNumbers: string;
    arabicText: string;
    translationEN: string;
    translationTR: string;
    theme: string;
  };
  dua: {
    id: string;
    title: string;
    category: string;
    audioUrl?: string;
  };
  story: {
    prophetName: string;
    episodeNumber: number;
    title: string;
    theme: string;
    audioUrl: string;
    isPremium: boolean;
  };
}
```

### Weekly Rotation Logic

Each mood has **7 verse options** → Rotate based on day of week (Monday = verse 1, Tuesday = verse 2, etc.)

```typescript
function getTodaysPathForMood(mood: MoodType): TodaysPathContent {
  const dayOfWeek = new Date().getDay(); // 0 = Sunday, 1 = Monday, etc.
  const verseIndex = dayOfWeek % 7; // Cycle through 7 verses

  const moodMapping = MOOD_CONTENT_MAP[mood];

  return {
    verse: moodMapping.verses[verseIndex],
    dua: moodMapping.duas[Math.floor(Math.random() * moodMapping.duas.length)],
    story: moodMapping.stories[0], // First story
  };
}
```

### Mood Change Logic

- **Weekly Check-in:** Every Monday, modal appears asking "How are you feeling this week?"
- **Manual Change:** User can tap mood indicator on Home screen anytime
- **Persistence:** Mood saved to AsyncStorage + Firestore (if authenticated)
